Política de Privacidad de Las Buenas Nuevas

Última actualización: 23/02/2025

El presente documento establece los términos en que la aplicación "Las Buenas Nuevas" (en adelante, "la aplicación") usa y protege la información proporcionada por sus usuarios. Nos comprometemos a garantizar la seguridad de los datos y solo utilizamos la información conforme a esta política. Nos reservamos el derecho de actualizar estos términos, por lo que recomendamos revisar esta página periódicamente.

1. Información que recopilamos

Nuestra aplicación recopila ciertos datos para mejorar la experiencia del usuario:

  • Firebase Analytics y Crashlytics: Datos de uso y fallos para mejorar la estabilidad.
  • Google AdMob: Anuncios en la aplicación que pueden recopilar datos de publicidad anónimos.
  • Correo electrónico: Si eliges contactarnos, tu dirección será usada solo para responderte.
  • Favoritos locales: Los usuarios pueden guardar publicaciones como favoritas; esta información se almacena solo en el dispositivo y se borra al desinstalar la app.

2. Datos compartidos con terceros

La aplicación usa servicios de terceros que pueden recopilar información:

  • Google Firebase (Analytics y Crashlytics)
  • Google AdMob (Publicidad personalizada)
  • Google Blogger API (para obtener publicaciones)
  • Google YouTube API (para mostrar videos)
  • Spotify y Anchor (para mostrar podcasts)

No compartimos datos con otros terceros fuera de Google.

3. Seguridad y cifrado

Todos los datos recopilados se cifran en tránsito utilizando los protocolos de seguridad de Google.

4. Eliminación de datos

Como la app no requiere cuentas de usuario, no almacenamos información personal. Si deseas solicitar la eliminación de algún dato asociado a Firebase Analytics o AdMob, contáctanos en:

📩 noreply.buenasnuevas@gmail.com

5. Cambios en la política

Nos reservamos el derecho de actualizar esta política en cualquier momento. Cualquier cambio será notificado dentro de la aplicación o en este documento.
